# Тема: Основні SQL команди. Агрегатні функції та групування

Даний репозиторій містить виконання домашнього завдання №3 з курсу "Relational Databases".

## Структура репозиторію

* `queries.txt` — текстовий файл із усіма SQL запитами.
* `screenshots/` — папка із скриншотами виконання запитів у MySQL Workbench.
* `README.md` — опис виконаної роботи.

## Опис виконаних завдань

### 1. Вибірка даних
* **Завдання 1.1:** Вибрано всі стовпчики з таблиці `products`.
* **Завдання 1.2:** Вибрано стовпчики `name` та `phone` з таблиці `shippers`.
* *Скриншоти:* `p1_1.jpg`, `p1_2.jpg`.

### 2. Агрегатні функції
Знайдено середнє (`AVG`), мінімальне (`MIN`) та максимальне (`MAX`) значення стовпчика `price` у таблиці `products`.
* *Скриншот:* `p_2.jpg`.

### 3. Унікальні значення та сортування
Вибрано унікальні пари `category_id` та `price`. Дані відсортовані за спаданням ціни та обмежені 10-ма рядками.
* *Скриншот:* `p_3.jpg`.

### 4. Фільтрація та підрахунок
Знайдено кількість рядків у таблиці `products`, де ціна знаходиться в межах від 20 до 100.
* *Скриншот:* `p_4.jpg`.

### 5. Групування даних
Знайдено кількість продуктів та середню ціну для кожного постачальника (`supplier_id`).
* *Скриншот:* `p_5.jpg`.

## Як запустити запити
Для перевірки запитів необхідно мати встановлену базу даних (схема `hw3`) та імпортовані дані з CSV файлів. Запити виконувались у середовищі **MySQL Workbench **.
